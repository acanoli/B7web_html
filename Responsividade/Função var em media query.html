Função var

A função var() é uma função CSS que permite usar valores dinâmicos em suas folhas de estilo. Ela é usada para acessar o valor de uma variável CSS definida anteriormente. As variáveis CSS são definidas usando a sintaxe --nome-da-variável: valor; e podem ser reutilizadas em todo o documento, facilitando a manutenção e a consistência do design.
